Guides

How to use Telegram E-commerce

Quick Start (Shop Owner)

  1. Create your bot via @BotFather and submit token.
  2. Add categories, then products with media, price, and stock.
  3. Add payment methods (number/QR/notes).
  4. Update welcome/support content and posters.
  5. Test checkout with payment proof and admin confirmation.
  6. Enable FAQ auto-replies and launch broadcasts.

Customer Journey

  1. Browse categories and product details.
  2. Select quantity, add to cart, or buy now.
  3. Complete profile (phone + shipping address).
  4. Choose payment method and transfer funds.
  5. Upload payment proof and wait for review.
  6. Receive order confirmation and track from My Orders.

Admin Daily Operations

  1. Review pending orders and payment proofs.
  2. Confirm or reject payments with status updates.
  3. Publish deals/news and broadcast campaigns.
  4. Maintain FAQ auto-reply buttons for support speed.
  5. Monitor stats and export reports regularly.
  6. Adjust plan/currency/settings as the shop scales.

Admin Documentation

Admin Panel 1 and Admin Panel 2 button explanations

These are practical explanations for bot owners and admins to understand what each button does and how to use it.

Admin Side - Admin Panel 1

These buttons are only available to the bot owner admin.

  1. Manage Orders - To manage and review customers' orders.
  2. Broadcast Message - To send a message to all customers who use your bot. Useful for promotion and announcement messages.
  3. Update Promo/News - To edit messages for the Promotion and News buttons.
  4. Update Caption - To change and edit captions of each page in the bot.
  5. Update Photo - To change the respective posters/photos of each page in the bot.
  6. Manage Admin - To add new admins or remove existing admins.

Two methods to add a new admin:

Forward Contact: Click this button and select the account you want to make an admin.

Generate Key: A random key is generated. Give this key to the target account, and they must paste and send it to your bot. Each key expires after 1 hour.

  1. Manage Blocked Users - To view and manage blocked users.

Admin Panel 2 Section

  1. Add Category - A category is required before adding products.

Difference between Category and Product:

Example 1: Samsung, iPhone, Xiaomi, Oppo, Vivo are products. Collectively they are in the Phone category.

Example 2: Computer is the category; Dell, HP, Acer are products.

You must create a category first before adding products.

  1. Add Product - After adding a category, add related products. Add at least one payment method first.

How to add a product:

  • Go to Add Product.
  • Select a category (or create one first).
  • Step 1: Send product name.
  • Step 2: Send price using English numbers only.
  • Step 3: Send product description/details, or skip.
  • Step 4: Send product photos/videos one by one, then press Finish.
  • Final Step: Press Save Product.

After saving, customers can see and purchase your product.

  1. Manage Payment - To manage payment accounts such as KBZPay and WavePay.
  2. Set Currency - To select transaction currency (for Myanmar, choose MMK).
  3. Manage Automation - To create FAQ buttons that auto-reply to customers. Create commands first, then link them.
  4. Manage Category/Product - To edit or delete category and product names.
  5. Set Up Commands - Create custom commands and auto-send prepared text/media.

What is a command? It is a slash command like /start, /share, /mmguid, /engguide.

How to set it up: If you want /info to auto-send contact details, add command name as info (without /), then send text/images and save.
